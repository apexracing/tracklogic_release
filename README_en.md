The built-in analysis engine automatically parses racing lines and input timing, generating clear summaries that instantly flag braking hesitation and line deviations.

## Quick Start

1. Download [Windows Installer](https://github.com/apexracing/tracklogic_release/releases/latest/download/apexracing-tracklogic-installer.exe)
2. Launch iRacing and enter any session
3. Double-click `TrackLogic.exe`
4. On first run, follow the iRacing OAuth login flow

## System Requirements

- **OS**: Windows 10 / 11 (64-bit)
- **Sim**: iRacing (installed and running)
- **Network**: Internet required for initial iRacing OAuth authentication
